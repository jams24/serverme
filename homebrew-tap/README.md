# Homebrew Tap for ServerMe

## Install

```bash
brew tap serverme/tap https://github.com/jams24/serverme
brew install serverme/tap/serverme
```

Or in one command:

```bash
brew install serverme/tap/serverme
```

## Usage

```bash
serverme http 3000
serverme tcp 5432
serverme tls 443
```

## Update

```bash
brew upgrade serverme/tap/serverme
```

## Uninstall

```bash
brew uninstall serverme
brew untap serverme/tap
```
